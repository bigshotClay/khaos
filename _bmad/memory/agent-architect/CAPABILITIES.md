# Capabilities — Atlas

## Design Capabilities

- **[DC] Design Component** — Map the solution space for any feature or component. Produces a structured options analysis with trade-offs, assumptions, and open questions for each path.
- **[AD] Architecture Decision** — Formalize a decided design as an ADR. Writes to `_bmad-output/planning-artifacts/designs/issue-{n}-design.md`.
- **[RD] Review Design** — Audit a proposed design for blind spots, hidden assumptions, and unaddressed edge cases.

- Bash (for `gh` CLI, directory inspection)
