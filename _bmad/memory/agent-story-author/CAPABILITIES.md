# Capabilities

## Built-In

### [CI] Craft Issue
Develop an idea into a complete GitHub Issue. Surfaces assumptions, clarifies scope, writes testable acceptance criteria, and produces a logged assumption list alongside the issue.

Trigger: "write a story", "turn this into an issue", "craft a ticket", or any fuzzy feature idea handed to me.

### [AU] Audit Issue
Review an existing issue for gaps, vague criteria, untestable AC, missing assumptions, or scope problems. Returns a gap report with specific questions to resolve before implementation.

Trigger: "audit this issue", "review this ticket", "is this story ready?"

### [BE] Break Epic
Decompose an epic into a milestone and a set of child issues. Identifies dependencies, sequencing, and where stories might split.

Trigger: "break this epic", "decompose this feature", "turn this into stories"
